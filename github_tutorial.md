## Khob chijori file ezafe konim?
baraye ezafe krdne file be in repo shoma dota rah darid
- manual
- Upload


shoma dar do halat bayad rooye add bezanid

![add](git_pics/add.png)



### **Option1 : Manual**
vaghty rooye add file mizanid, agar gozine **create new file** bezanid baratoon yek ja baz mishe k mitoni esme file bezarid va dakhelesh ya type konid ya copy paste konid va dar enthea **commit** konid, commit hamon save hast

![manual](git_pics/manual.png)


### **Option2 : Upload**
dovomin gozine zamani hast k shoma file ro darid va mikhahid kole file ro upload konid yekja . bejaye gozienye **create new file** rouye **upload file** bezanid

inja rahat ya rooye **choose your file** bezanid ya fileton ro bekeshid vasate safe , badesh rooye commit bezanid ta uplaod beshe

![upload](git_pics/upload.png)

### Collaborator (Hamkar)
github behetoon ejaze mide ke yek nafar dige ro ham b oon **repository** k roosh kar mikonid ezafe konid ta baham dige taghirat ro anjam bedid

khob rooye porozhaton shoma bad az inke **github** sakhtid , bad az inke **repository** sakhtid, bad azinke hame file ro rooye **repo** gozashtid va vaghty dashtid taghirat ro anjam midadid , age man ham dastresi dashte basham b repoye shoma , mitonam biam va baratoon harja eshtebah mikonid taghir bedam va comment bezaram, ya agar komak khastid balaye file benevisid man rahnamaeeton konam

baraye inkar shoma bayad accounte man ro **collaborator** konid


chegoone? aval berid safeye [home](https://github.com/) va rooye oon **repository** delkhahetoon click konid va sepas k vared shodid rooye stting bznid
